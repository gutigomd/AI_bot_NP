# Napas Project AI Bot (MVP)

Интеллектуальный ИИ-ассистент для платформы Napas Project, созданный для помощи пользователям в навигации по сайту. Бот отвечает на вопросы по правилам, инвентарю, жалобам и банам, используя актуальный контент страниц, выкачанный через Jina AI.

---

## 📸 Демонстрация работы

Вот так выглядит и работает готовый прототип прямо сейчас:

### 1. Виджет чата на сайте
При переходе на страницу в правом нижнем углу появляется минималистичная кнопка вызова ассистента.

### 2. Интерактивный диалог с ИИ
При клике открывается окно чата. Бот использует модель **Gemini-2.5-Flash** и отвечает строго по базе знаний сайта (без «воды» и галлюцинаций), подстраиваясь под язык пользователя.
